# C++ Currency Denomination Calculator

A beginner-friendly C++ program that converts a given dollar amount into the minimum number of currency bills.  
This project highlights arithmetic operations, modulus usage, and clear console-based output formatting.

---

## Features
- Accepts a total dollar amount as input  
- Calculates number of $20, $10, $5, and $1 bills  
- Uses integer division and modulus for accurate breakdown  
- Displays results in a structured format  

---

## Concepts Used
- Integer division and modulus (`/`, `%`)  
- Arithmetic operations  
- Basic input/output handling  
- Sequential computation logic  

---

## How to Run
1. Save the program as `main.cpp`  
2. Compile using:  
   ```bash
   g++ main.cpp -o denomination_calculator
